# 🌌 Solar System Simulation (OpenGL + Python)

A real-time **3D Solar System simulation** built using **Python, Pygame, and PyOpenGL**.  
This project visualizes planets, moons, orbital motion, and a dynamic camera system in an interactive space environment.

---

## 🚀 Features

- 🪐 Realistic 3D solar system simulation
- 🌍 All major planets + Moon system
- 🌞 Emissive Sun lighting effect
- 🛰️ Orbital motion with realistic speed scaling
- 🌠 Starfield background (1000+ stars)
- 🎥 Free camera movement (mouse + keyboard control)
- 🪐 Saturn rings rendering
- ⏯️ Pause / Resume simulation
- 🔭 Toggle orbit visibility

---

## 🎮 Controls

| Key / Mouse | Action |
|-------------|--------|
| Mouse Drag | Rotate camera |
| W / S / ↑ ↓ | Zoom in / out |
| A / D / ← → | Move target left/right |
| SPACE | Pause / Resume simulation |
| O | Toggle planet orbits |
| ESC | Exit |
